request system external-list show type {ip|name|url} name <name-of-the-list>: This command is used to display the contents of an External Dynamic List. You need to specify the type of list (IP, name, or URL) and provide the name of the list you want to view.

request system external-list refresh type {ip|name|url} name <name-of-the-list>: This command is used to refresh the contents of an External Dynamic List. It updates the entries in the list based on the latest information from the external source. Again, you need to specify the type of list (IP, name, or URL) and provide the name of the list you want to refresh.
